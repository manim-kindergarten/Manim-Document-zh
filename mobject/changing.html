Changing¶

AnimatedBoundary¶

class manimlib.mobject.changing.AnimatedBoundary(vmobject, **kwargs)¶

动态变化的边界

传入需要显示动态边界的物体 vmobject

  • colors 表示变化中出现的颜色

  • max_stroke_width 表示边界最大的粗细

  • cycle_rate 表示循环率

TracedPath¶

class manimlib.mobject.changing.TracedPath(traced_point_func, **kwargs)¶

记录路径的VMobject

传入一个可调用的对象 traced_point_func (一般为 mob.get_center() )

  • min_distance_to_new_point : 两点之间的最小距离，若小于此距离则不增加点
